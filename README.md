# Якушев - Кухни на заказ | Premium Landing Page

Премиальный одностраничный сайт-визитка для производства кухонь на заказ с современным дизайном и детальной проработкой UI/UX.

## Технологии

- **React 18** - современная библиотека для построения UI
- **TypeScript** - типизация для надежного кода
- **Vite** - быстрый сборщик и dev-сервер
- **CSS Modules** - изолированные стили компонентов
- **Swiper** - производительный слайдер с поддержкой touch-событий
- **Lucide React** - современные SVG иконки
- **Fontsource** - self-hosted премиум шрифты

## Дизайн

### Типографика
- **Playfair Display Variable** - элегантный serif для заголовков
- **Inter Variable** - современный sans-serif для текста
- Fluid typography с clamp() для адаптивности
- Продуманная типографическая иерархия

### Визуальный стиль
- Черный фон с subtle эффектами (градиенты, текстура)
- Золотой акцент (#d4af37) для премиум-ощущения
- Система spacing scale (8px base)
- CSS-переменные для консистентности
- Тонкие свечения и тени

## Особенности

- **Полностью адаптивный дизайн** (desktop, tablet, mobile)
- **Якорная навигация** для быстрого скроллинга между секциями
- **Плавные анимации** с Intersection Observer
- **Премиум UI-компоненты:**
  - Карточки типов кухонь (современные, классические, лофт, и др.)
  - Вертикальный таймлайн процесса создания кухни
  - Бейджи преимуществ (фурнитура Blum, гарантия 3 года)
  - CTA-блок с призывом создать кухню мечты
- **Lightbox** для просмотра фотографий готовых проектов
- **Lazy loading** изображений
- **Accessibility** (ARIA, фокус-стили, семантика)
- **SEO-оптимизация** (мета-теги, Open Graph для кухонного бизнеса)

## Установка и запуск

### Требования

- Node.js >= 18.0.0
- npm >= 9.0.0 (или yarn/pnpm)

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу: `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

### Предварительный просмотр production-сборки

```bash
npm run preview
```

### Линтинг

```bash
npm run lint
```

## Настройка контактов

Чтобы изменить ссылки на соцсети и контактную информацию, отредактируйте файл:

```
src/config/contacts.ts
```

Там находятся:
- `socialLinks` - массив ссылок на соцсети (Telegram, WhatsApp, VK)
- `contactInfo` - телефон и email

## Добавление/удаление изображений

1. Поместите изображения в папку `public/assets/`
2. Отредактируйте массив `images` в файле `src/components/WorksSlider/WorksSlider.tsx`

## Deployment на GitHub Pages

Проект настроен для автоматического деплоя на GitHub Pages через GitHub Actions.

### Настройка (один раз):

1. Создайте репозиторий на GitHub
2. Перейдите в Settings → Pages
3. В разделе "Source" выберите **GitHub Actions**

### Деплой:

```bash
# Сделайте commit и push в main ветку
git add .
git commit -m "Deploy to GitHub Pages"
git push origin main
```

GitHub Actions автоматически:
- Соберет проект (`npm run build`)
- Задеплоит на GitHub Pages
- Сайт будет доступен по адресу: `https://[username].github.io/[repo-name]`

### Локальная сборка:

```bash
npm run build     # Собрать проект
npm run preview   # Проверить prod-сборку локально
```

### Альтернативные варианты деплоя:

- **Vercel**: `vercel --prod`
- **Netlify**: подключите репозиторий, build command: `npm run build`, publish directory: `dist`
- **Любой VPS/хостинг**: загрузите содержимое папки `dist/` после сборки
